# Sihelelwe Mchobokazi | Portfolio

A modern, responsive portfolio website built with Next.js 16, TypeScript, and Tailwind CSS. This portfolio showcases my projects, technical skills, and professional background as a full-stack developer.

## Live Demo

Visit the deployed application at:  
**[https://wdd430-portfolio.vercel.app](https://wdd430-portfolio.vercel.app)**

---

## Project Overview

This portfolio was built as part of the WDD430 course to demonstrate:
- Next.js App Router with Server Components
- Component composition and reusability
- TypeScript for type safety
- Tailwind CSS for styling
- File-based routing
- API routes

### Features
- **Header** – Navigation links with Next.js Link for client-side transitions
- **Project List** – Dynamic rendering of project cards
- **Project Card** – Displays title, description, technologies, and external links
- **About Page** – Personal bio and technical skills with custom SkillCard component
- **Footer** – Dynamic copyright year
- **API Route** – `/api/hello` returns a JSON response
- **Responsive Design** – Mobile-first grid layout
